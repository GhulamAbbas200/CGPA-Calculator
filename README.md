# MAJU CGPA Calculator 🎓

A fast, responsive, and user-friendly GPA and CGPA calculator built specifically for the grading policies of **Muhammad Ali Jinnah University (MAJU)**. 

This tool helps students easily estimate their Semester GPA (SGPA) and Cumulative GPA (CGPA) by entering their course marks or selecting letter grades directly.

## ✨ Features

* **Program-Specific Grading:** Toggle between **Undergraduate** and **Graduate** grading schemes. The calculator automatically adjusts boundaries (e.g., handling 'D' grades for Undergrads, which count as 'F' for Graduates).
* **Dual Input Modes:** Calculate your GPA by entering exact **Marks (0-100)** or by selecting a **Letter Grade** from the dropdown. 
* **Real-time Calculations:** SGPA and points are calculated instantly as you type.
* **CGPA Integration:** Enter your previous credit hours and historical CGPA to see how your current semester impacts your overall standing.
* **Built-in Policy Reference:** A handy drop-down reference table allows users to double-check the official university grading brackets at any time.
* **Fully Responsive:** Optimized for both desktop and mobile screens.

## 🛠️ Technologies Used

* **React** - UI framework
* **Vite** - Build tool and development server
* **Tailwind CSS (v4)** - Utility-first CSS framework for styling
* **Lucide React** - Beautiful and consistent iconography
